## ✨ About the Project

**Graphitie Originals** – *Designing seamless digital experiences*
*Our design partner creates UI/UX frameworks and product design for web and app projects.*

This SPA is a **tech showcase** integrating the design and UX provided by Graphitie Originals.

Project highlights:

* **Design Partner:** Graphitie Originals provides UI/UX and product design frameworks.
* **SPA Architecture:** Single-scroll, section-mounted, dynamically loaded from modular HTML fragments.
* **Tech Integration:** Frontend uses vanilla JS, Bootstrap, GSAP, with assets from Contentful and ImageKit.

## 📁 Folder Structure

```
graphitie/
├── index.html       # SPA root shell
├── css/             # Section-specific and main CSS
├── js/              # Section mounting & app logic

├── assets/          # Images, videos, icons managed via Contentful
├── README.md        # Project documentation
```

## 🚀 Getting Started

**1. Clone the repository**

```bash
git clone https://github.com/codepagehere/graphitie.git
cd graphitie
```

**2. Configure environment variables**

```bash
cp .env.example .env
# Edit .env with your settings for Contentful / ImageKit
```

**3. Install dependencies**

```bash
npm install
```

**4. Run development server**

```bash
# Start the local SPA server with live reload
graphitie.bat
```
